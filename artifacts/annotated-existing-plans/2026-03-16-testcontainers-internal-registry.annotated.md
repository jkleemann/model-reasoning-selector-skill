> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

- [ ] **Step 3: Add the shared helper**

```kotlin
package com.dvag.testcontainer

import org.testcontainers.utility.DockerImageName

object DvagDockerImages {
    fun postgres(): DockerImageName =
        DockerImageName.parse("dvagcr.azurecr.io/platform/extern/postgres:16")
            .asCompatibleSubstituteFor("postgres")

    fun mssql(): DockerImageName =
        DockerImageName.parse("dvagcr.azurecr.io/platform/extern/mssql:2022-latest")
            .asCompatibleSubstituteFor("mcr.microsoft.com/mssql/server")
}
```
